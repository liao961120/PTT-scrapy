PTT 用語

關於

這裡提供批踢踢常見用語的查找與檔案。常見用語自 PTT 鄉民百科 爬取，有 5 種類別：

  1. title: HTML h1.page-header__title 之下的文字
  2. bold：HTML b (粗體)之下的文字
  3. link_title：含有 title attribute 的 a tag
  4. link_new：HTML a.new 之下的文字
  5. bracket：「、」 之間的文字

資料下載

  • dict.json: 自網頁爬取的原始資料，內容最為豐富
  • ptt-terms.tsv: 整理過後之資料，與此網頁上的資料相同
  • 原始碼 (GitHub)
